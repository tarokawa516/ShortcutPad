# 開発ガイド

## 必要な環境

- Windows
- Visual Studio（.NETデスクトップ開発）または.NET 10 SDK

外部NuGetパッケージは現在使用していません。

## ビルド

リポジトリのルートで実行します。

```powershell
dotnet build ShortcutPad/ShortcutPad.csproj
```

## 実行

```powershell
dotnet run --project ShortcutPad/ShortcutPad.csproj
```

## 手動確認

変更後は少なくとも次を確認します。

1. メモ帳などへ複数行のテキストを入力する。
2. テキスト入力側にフォーカスがある状態でShortcutPadの「全選択」を押す。
3. 対象テキストが選択され、ShortcutPad側へフォーカスが移らないことを確認する。
4. 「全選択 → コピー」と「貼り付け」を確認する。
5. 切り取り、元に戻す、検索を確認する。
6. パネルの移動、折り畳み、終了を確認する。
7. Windowsの表示倍率を変更した環境でもレイアウトが崩れないことを確認する。
8. 可能であればSplashtop経由のタッチ操作でも確認する。

## 権限に関する確認

通常権限のShortcutPadから、管理者権限で起動されたアプリへは入力を送れない場合があります。これはWindowsの入力保護によるものです。問題の切り分け時には、送信先とShortcutPadの実行権限を確認してください。
